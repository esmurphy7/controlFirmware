These UDP packets are sent over WiFi on the ‘titanoboa’ network over port 12345. 
Currently the snake sends them in order (type 1, 2, 3, 4, 5) on a period of 
approximately 78ms. This might change in the future; we might want to prioritize 
important packets.


// Packet Name: Head and Module General Info
// Packet Type: 1
// Total Size: 128 bytes

Byte0:  Packet Type = 1 8-bit
Byte1:  Head, Number of Modules Connected 8-bit
Byte2:  Head, Battery Voltage 16-bit unsigned (MSB)
Byte3:  Head, Battery Voltage 16-bit unsigned
Byte4:  Module 1, Battery Voltage 16-bit unsigned (MSB)
Byte5:  Module 1, Battery Voltage 16-bit unsigned
Byte6:  Module 1, Motor Speed 16-bit unsigned (MSB)
Byte7:  Module 1, Motor Speed 16-bit unsigned
Byte8:  Module 1, Pressure Sensor 16-bit unsigned (MSB)
Byte9:  Module 1, Pressure Sensor 16-bit unsigned
..
.. Repeat Module X, 6 Bytes per Module, 6 Modules
..
Byte34:  Module 6, Battery Voltage 16-bit unsigned (MSB)
Byte35:  Module 6, Battery Voltage 16-bit unsigned
Byte36:  Module 6, Motor Speed 16-bit unsigned (MSB)
Byte37:  Module 6, Motor Speed 16-bit unsigned
Byte38:  Module 6, Pressure Sensor 16-bit unsigned (MSB)
Byte39:  Module 6, Pressure Sensor 16-bit unsigned
Byte40:  Spare
..
..
Byte123:  Spare
Byte124:  Spare



// Packet Name: Angles and Setpoints
// Packet Type: 2
// Total Size: 128 bytes

Byte0:  Packet Type = 2 8-bit
Byte1:  Timestamp millis() (MSB)
Byte2:  Timestamp millis()
Byte3:  Timestamp millis()
Byte4:  Timestamp millis()
Byte5:  Spare
Byte6:  Spare
Byte7:  Module 1, Vertibrae 1, Horizontal Setpoint Angle 8-bit
Byte8:  Module 1, Vertibrae 1, Horizontal Angle 8-bit
Byte9:  Module 1, Vertibrae 1, Vertical Setpoint Angle 8-bit
Byte10:  Module 1, Vertibrae 1, Vertical Angle 8-bit
..
.. Repeat Module X, Vertibrae Y, 4 Bytes per Vertibrae, 30 Vertibrae
..
Byte127:  Module 6, Vertibrae 5, Vertical Angle 8-bit
